# library-seat-system

## 目录版本说明

### 新版目录

- `seatmanage/`：新版移动端代码，本次新增上传，优先查看这个目录。
- `new-pc-main/`：新版 PC/Web/服务端整合代码。

### 旧版目录

- `library-seat-manage/`：旧版移动端代码，当前保留用于对照。
- `new-pc-main/library-seat-manage/`：旧版移动端代码副本，保留在 PC 项目目录中。

### 补充说明

- 当前仓库同时保留新旧版本，便于对比、迁移和继续开发。
- 如果你只看最新移动端，请使用 `seatmanage/`。
- 如果你只看最新 PC/Web 端，请使用 `new-pc-main/`。
- `seatmanage/` 目录中保留了源目录自带的 `new-pc-main/` 子目录，该内容按原目录一并上传。
